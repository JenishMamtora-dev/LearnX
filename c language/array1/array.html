What is an Array?

An Array in C language is used to store multiple values in a single variable, instead of declaring separate variables for each value. To create an array, define the data type (like int) and specify the name of the array followed by square brackets [].
There are three types of array in c programming:One DIMENSIONAL,Two DIMENSIONAL and Multi DIMENSIONAL
