Nested-If statement

Definition

It conists of if condition inside another if statement .

Syntax

if (condition) 
{
   if (condition)
    {
     
  // block of code to be executed if both the condition is true.
                } 
                else
                 {
  // block of code to be executed if second  if  condition is false
}
}
else
 {
  // block of code to be executed if the condition is false.
}

Key Points

  • It is used when multiple condition is to be checked

  • It first check the if condition

  • If condition is true than it checks the inner if condition and if it is true than the block of code is executed.

  • otherwise it will execute the else part of inner if statement.

  • If first if condition is false than it directly executes the else part of outer if statement.

  • It is called Nestedif statement.
