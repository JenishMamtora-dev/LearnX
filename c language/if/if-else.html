If-Else statement

Definition

The if-else statement checks the if condition and execute the block of code otherwise execute the else part.

Syntax

if (condition) 
{
  // block of code to be executed if the condition is true
}
else
 {
  // block of code to be executed if the condition is false.
}

Key Points

  • It is used when only a single condition is to be checked

  • No condition had to written in the else part.

  • It first check the if statement and executethe block of code if condition is true.

  • otherwise it will directly execute the else part.

  • It is called if else statement.
