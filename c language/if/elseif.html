Else-if Ladder

Definition

It conists of multiple if else if condition.

Syntax

if (condition1) 
{
 // code to be executed if condition1 is true
} 
else if (condition2)
 {
    // code to be executed if condition1 is false and condition2 is true
} 
else if (condition3)
 {
    // code to be executed if condition1 and condition2 are false, and condition3 is true
}
// ... (more else if clauses can be added)
else
{
    // code to be executed if all preceding conditions are false (optional)
}

Key Points

  • It is used when multiple condition is to be checked

  • It first check the if condition

  • If condition is true than block of code is executed.

  • If condition is false it check the 2nd condition and if that is true its block of code is been executed .

  • Similarly it check all the else if condition and execute the block of code when condition is true.

  • if no condition is true it will execute the else part.

  • It is called else if Ladder.

Example

    #include <stdio.h>
    void main()
    {
        int mark; 
        clrscr();
        printf("enter mark:");

        scanf("%d",&mark);
        if (mark >= 90 && mark <= 100)
        {
             printf("Grade: A (Excellent)\n");
        } 
        else if (mark >= 80 && mark < 90) 
        {
            printf("Grade: B (Very Good)\n");
        } 
        else if (mark >= 70 && mark < 80) 
        {
            printf("Grade: C (Good)\n");
        }
        else if (mark >= 60 && mark < 70)
         {
            printf("Grade: D (Pass)\n");
        } 
        else if (mark >= 0 && mark < 60)
        {
             printf("Grade: F (Fail)\n");
        }
        else
        {
             printf("Invalid score entered.
             Please enter a score between 0 and 100.\n");
        } 
    getch();
    }
