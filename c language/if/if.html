If statement

Definition

The if statement executes the block of code if the given conditionis true.

Syntax

if (condition) 
{
  // block of code to be executed if the condition is true
}

Key Points

  • It is used when only a single condition is to be checked

  • It will not execute the block of code if condition is false.

  • It is called Simple if statement.
