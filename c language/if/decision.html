C Decision-Making

What is Decision-Making Statement?

In C programming, decision-making statements, also known as conditional statements,
are control flow statements that allow the program to execute different blocks of code based
on whether a specified condition evaluates to true or false.
There are four types of Decision-Making statement in c programming:if,if-else,nested ifand else if ladder
