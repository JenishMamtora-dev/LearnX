Return statement

Definition

The Return statement is used to terminate the execution of a function and return control to the calling function.

Syntax

        return expression;

Key Points

  • It is terminated using semi colon(;).

  • It is be used with user defined function.

  • It can optionally return a value to the caller.

  • When return is executed the control will again go to main function.
