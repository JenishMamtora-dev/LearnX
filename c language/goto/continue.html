Continue statement

Definition

The Continue statement is used the current iteration of the loop is skipped, and the control flow proceeds to the next iteration of the loop.

Syntax

        continue;

Key Points

  • It is terminated using semi colon(;).

  • It can only be used with loops .

  • When used in loop it will skip that particular iteration.

  • Continue statement will not terminate the loop when condition is satisfied.

Example

    #include <stdio.h>
    void main(){
       int i=0;
       clrscr();
        for ( i = 0; i < 10; i++)
         {
            if (i == 5) 
            {
                 continue; // Exits the loop when i is 5
            }
         printf(\n"%d ", i);
    }
    getch();
}

    
Output:
        0
        1
        2
        3
        4
        6
        7
        8
        9
