Break statement

Definition

The Break statement is used to terminate the innermost loop or switch case statement.

Syntax

        break;

Key Points

  • It is terminated using semi colon(;).

  • It can only be used with loops & switch case.

  • When used in multiple loop it will always terminate the inner loop.

  • Break statement will terminate the loop when condition is satisfied.

Example

    #include <stdio.h>
    void main(){
       int i=0;
       clrscr();
        for ( i = 0; i < 10; i++)
         {
            if (i == 5) 
            {
                 break; // Exits the loop when i is 5
            }
         printf(\n"%d ", i);
    }
    getch();
}

    
Output:
        0
        1
        2
        3
        4
