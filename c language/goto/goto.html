Goto statement

Definition

The goto statement in C is a control statement used to transfer the flow of execution to a specific labeled statement in the same function. It is often called an unconditional jump statement.

Syntax

       goto label;

Key Points

  • label is a user-defined identifier followed by a colon :.

  • It can jump forward or backward in the code.

  • goto label; transfers control to the statement after label:.

  • Note: goto makes the program less readable and harder to debug that's why it is not recommended in modern coding.

Example

    #include <stdio.h>
    void main(){
       int num = 1;
       clrscr();
    start: // label
    printf("%d\n", num);
    num++;

    if (num <= 5)
    {
        goto start; // jumps back to label
    }
    getch();
}

    
Output:
        
        1
        2
        3
        4
        5
