Variables in C

Definition

Variables are containers for storing data values. In C, every variable must be declared with a specific data type before use.

Syntax

data_type variable_name = value;

Key Points

  • Variables must be declared before they are used.
  • Each variable has a type (int, float, char, etc.).
  • Multiple variables of the same type can be declared together.
  • Initialization can be done at the time of declaration.

Example

#include 
int main() {
    int age = 20;
    float salary = 25000.50;
    char grade = 'A';

    printf("Age: %d\n", age);
    printf("Salary: %.2f\n", salary);
    printf("Grade: %c\n", grade);

    return 0;
}
