C Loops

What are Loops?

In C programming, loops are constructs that repeatedly execute a block of code as long as a condition is true. They help you run repetitive tasks without duplicating code. The three main types of loops in C are: for loop, while loop, and do-while loop.

Loop Types

For Loop

Use when you know how many times you want to repeat.

While Loop

Repeat until a condition becomes false.

Do-While Loop

Executes at least once, then repeats while condition is true.
