C For Loop

Definition

A for loop is used when the number of iterations is known. It executes a block of code repeatedly until the condition becomes false.

Syntax

for(initialization; condition; increment/decrement) {
    // code to be executed
}

Key Points

  • Initialization runs once at the beginning.
  • Condition is checked before each iteration.
  • Increment/Decrement updates the loop variable.

Example

#include <stdio.h>

void main() {
    int i;
    for(i = 1; i <= 5; i++) {
        printf("%d\n", i);
    }
}

Output:
    1
    2
    3 
    4
    5
