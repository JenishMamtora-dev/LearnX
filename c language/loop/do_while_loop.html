C Do While Loop

Definition

A do while loop is similar to a while loop, but the code executes at least once before checking the condition.

Syntax

do {
    // code to execute
} while(condition);

Key Points

  • The code runs at least once.
  • The condition is checked after execution.
  • Useful when one-time execution is required before checking.

Example

#include <stdio.h>

void main() {
    int i = 1;
    do {
        printf("%d\n", i);
        i++;
    } while(i <= 5);
}

Output:
    1
    2
    3 
    4
    5
