Syntax

switch (expression)
 {
     case value1:
        // statements
        break;

     case value2:
        // statements
        break;
        ...

    default:
        // statements (if no case matches)
}

Key Points

  • The expression inside switch is evaluated once.

  • Its result is compared with each case value.

  • After the case In a switch statement, each case must be followed by a colon (:).

  • If a match is found, the corresponding block executes.

  • The break statement stops execution from “falling through” to the next case.

  • default is optional it runs if no case matches.
