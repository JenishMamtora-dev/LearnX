Syntax

union union_name 
{
    data_type member1;
    data_type member2;
    ...
    data_type memberN;
};

Key Points

  • union is declared using the keyword union.

  • During Union declaration it is neccessary to terminate it with semi colon(;)

  • All members share the same memory, so the size of the union equals the size of its largest member.

  • Only one member can store a value at a time.

  • Useful for saving memory when a variable may hold different types at different times..

  • Access members using the dot operator (.).

  • Unions are commonly used in embedded systems or memory-efficient programs.
