Syntax

struct structure_name 
{
    data_type member1;
    data_type member2;
    ...
    data_type memberN;
};

Key Points

  • struct is the keyword to define a structure.

  • During Structure declaration it is neccessary to terminate it with semi colon(;)

  • A structure can have different data types inside it.

  • To access structure members, we use the dot operator (.).

  • Memory is allocated separately for each member.

  • Structures can be nested (a structure inside another structure).

  • They are commonly used to group related data (like student details: roll number, name, marks).
