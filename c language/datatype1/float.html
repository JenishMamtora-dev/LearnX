Float

Definition

The floating datatype in C is used to store the precision floating-point values.

Syntax

               float variable_name;

Key Points

  • The Float datatype is declared in C using Float keyword.

  • It can store 4 bytes of data.

  • The format specifer of integer is %f.

  • The Float can store decimal values and display .00 if only integer is entered.

Example

    #include <stdio.h>
    void main()
    {
    float i=10.10;
    clrscr();       
    printf("\n Value of i is: %.2f",i)   
    getch();
    }

    
Output:
        value of i is:10.10
