Double

Definition

Double datatype stores fractional numbers, containing one or more decimals.Sufficient for storing 15 decimal digits.

Syntax

               double variable_name;

Key Points

  • The double datatype is declared in C using double keyword.

  • It can store 8 bytes of data.

  • The format specifer of double is %lf.

  • The double datatpe works same as float but it can store more data than float.

Example

    #include <stdio.h>
    void main()
    {
    double i=10.101010;
    clrscr();       
    printf("\n Value of i is: %lf",i)   
    getch();
    }

    
Output:
        value of i is:10.101010
