Integer

Definition

The integer datatype in C is used to store the integer numbers
(any number including positive, negative and zero without decimal part).

Syntax

                int variable_name;

Key Points

  • The Integer datatype is declared in C using int keyword.

  • It can store 2 or 4 bytes of data.

  • The format specifer of integer is %d.

  • The range of integer datatype is from -2,147,483,648 to 2,147,483,647.

Example

    #include <stdio.h>
    void main()
    {
    int i=10;
    clrscr();       
    printf("\n Value of i is: %d",i)   
    getch();
    }

    
Output:
    value of i is:10
