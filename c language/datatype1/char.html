Character

Definition

The character datatype Stores a single character/letter/number, or ASCII values .

Syntax

                char variable_name;

Key Points

  • The Character datatype is declared in C using char keyword.

  • It can store 1 bytes of data.

  • The format specifer of character is %c.

  • We can also store strings using char & %s is the format specifier used for it.
