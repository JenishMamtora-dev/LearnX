CSS Foundations The Building Blocks

Master these essential concepts to build a solid CSS foundation.

Selectors & Specificity

Understand how browsers determine which styles apply

/* Element selector */
p { color: #fff; }

/* Class selector */
.text-primary { color: #4CAF50; }

/* ID selector */
#header { background: #121212; }

Box Model

The fundamental layout concept in CSS

.box {
    width: 300px;
    padding: 20px;
    border: 2px solid #4CAF50;
    margin: 10px;
    box-sizing: border-box;
}

Display & Positioning

Control how elements flow in your layout

.container {
    display: flex;
    justify-content: center;
    align-items: center;
}

.fixed-item {
    position: fixed;
    top: 0;
    right: 0;
}

Transitions & Animations

Bring your interfaces to life with motion

.animated-button {
    transition: all 0.3s ease-out;
}

.animated-button:hover {
    transform: scale(1.05);
    box-shadow: 0 5px 15px rgba(76, 175, 80, 0.3);
}

Responsive Design

Create layouts that adapt to any screen size

/* Mobile first approach */
.container {
    padding: 1rem;
}

@media (min-width: 640px) {
    .container {
        padding: 2rem;
    }
}

@media (min-width: 1024px) {
    .container {
        max-width: 1200px;
        margin: 0 auto;
    }
}

Advanced CSS Magic Push the Boundaries

Cutting-edge techniques for creating truly exceptional designs.

CSS Variables

Create dynamic, maintainable styles with custom properties

:root {
    --primary-color: #4CAF50;
    --bg-dark: #121212;
    --transition-speed: 0.3s;
}

.component {
    background: var(--bg-dark);
    color: var(--primary-color);
    transition: all var(--transition-speed) ease;
}

Clip-Path & Shapes

Break out of rectangular boxes with creative shapes

.speech-bubble {
    clip-path: polygon(
        0% 0%,
        100% 0%,
        100% 75%,
        75% 75%,
        75% 100%,
        50% 75%,
        0% 75%
    );
    shape-outside: polygon(
        0% 0%,
        100% 0%,
        100% 75%,
        75% 75%,
        75% 100%,
        50% 75%,
        0% 75%
    );
}

Blend Modes & Filters

Create stunning visual effects directly in CSS

.hero-image {
    background-image: linear-gradient(
        to right,
        #4CAF50,
        rgba(0, 0, 0, 0.7)
    ), url('hero.jpg');
    background-blend-mode: multiply;
    filter: drop-shadow(0 4px 8px rgba(0, 0, 0, 0.3));
}

Custom Properties & Theming

Create dynamic themes and style variations

[data-theme="dark"] {
    --bg-color: #121212;
    --text-color: #e0e0e0;
    --accent-color: #4CAF50;
}

[data-theme="light"] {
    --bg-color: #ffffff;
    --text-color: #333333;
    --accent-color: #2E7D32;
}
