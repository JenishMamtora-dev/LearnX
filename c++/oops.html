OOP Concepts in C++

Definition

Object-Oriented Programming (OOP) is a programming paradigm based on objects and classes.

Key Points

  • Encapsulates data and functions in objects.
  • Supports code reusability via inheritance.
  • Polymorphism allows same function name with different behavior.
  • Abstraction hides internal details from user.
  • OOP improves modularity, readability, and maintainability of code.

Types / Features

  • Class: Blueprint for objects.
  • Object: Instance of a class.
  • Inheritance: Deriving new class from existing class.
  • Polymorphism: Function overloading and overriding.
  • Encapsulation: Wrapping data & functions in class with access specifiers.
  • Abstraction: Hiding internal implementation using abstract classes or interfaces.

Syntax

// Class syntax
class ClassName {
private:
    // private members
public:
    // public members
};

// Object syntax
ClassName obj;
obj.memberFunction();

Examples

1. Class & Object

#include <iostream>
using namespace std;

class Car {
public:
    string brand;
    void display() { cout << "Brand: " << brand; }
};

int main() {
    Car car1;
    car1.brand = "BMW";
    car1.display();
}
Brand: BMW

2. Inheritance

#include <iostream>
using namespace std;

class Vehicle {
public:
    void display() { cout << "This is a vehicle."; }
};

class Car : public Vehicle {
};

int main() {
    Car c;
    c.display();
}
This is a vehicle.

3. Polymorphism (Function Overloading)

#include <iostream>
using namespace std;

class Math {
public:
    int add(int a,int b) { return a+b; }
    double add(double a,double b) { return a+b; }
};

int main() {
    Math m;
    cout << m.add(5,3) << " ";
    cout << m.add(2.5,3.5);
}
8 6

4. Encapsulation

#include <iostream>
using namespace std;

class Bank {
private:
    int balance;
public:
    void setBalance(int b) { balance = b; }
    int getBalance() { return balance; }
};

int main() {
    Bank b;
    b.setBalance(1000);
    cout << "Balance: " << b.getBalance();
}
Balance: 1000

5. Abstraction (Using Abstract Class)

#include <iostream>
using namespace std;

class Shape {
public:
    virtual void draw() = 0; // pure virtual function
};

class Circle : public Shape {
public:
    void draw() { cout << "Drawing Circle"; }
};

int main() {
    Circle c;
    c.draw();
}
Drawing Circle
