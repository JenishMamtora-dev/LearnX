Arrays & Strings — C++

Definition

An array is a collection of elements of the same type stored in contiguous memory. A string is a sequence of characters terminated with a null character ('\0') or managed by the string class.

Arrays

1. One-Dimensional Array

#include <iostream>
using namespace std;

int main() {
    int arr[3] = {10,20,30};
    for(int i=0;i<3;i++)
        cout << arr[i] << " ";
}
10 20 30

2. Two-Dimensional Array

#include <iostream>
using namespace std;

int main() {
    int matrix[2][3] = {{1,2,3},{4,5,6}};
    for(int i=0;i<2;i++) {
        for(int j=0;j<3;j++) {
            cout << matrix[i][j] << " ";
        }
        cout << endl;
    }
}
1 2 3
4 5 6

3. Array Initialization Methods

int a[5] = {1,2,3};    // remaining values = 0
int b[] = {10,20,30};  // size auto-detected

Strings

1. C-Style String (char array)

#include <iostream>
using namespace std;

int main() {
    char name[] = "C++";
    cout << name;
}
C++

2. String Functions (C library)

#include <iostream>
#include <cstring>
using namespace std;

int main() {
    char str1[20] = "Hello";
    char str2[20] = "World";
    strcat(str1, str2); // join
    cout << "Length: " << strlen(str1) << endl;
    cout << "Concat: " << str1;
}
Length: 10
Concat: HelloWorld

3. C++ String Class

#include <iostream>
#include <string>
using namespace std;

int main() {
    string s = "Hello";
    s.append(" C++");
    cout << "Length: " << s.length() << endl;
    cout << "Substring: " << s.substr(0,5);
}
Length: 9
Substring: Hello
