Templates in C++

Definition

Templates allow writing generic functions or classes that work with any data type.

Key Points

  • Two types: Function Templates and Class Templates.
  • Enable code reusability.
  • Type is specified at compile-time.

Syntax

Function Template

template <typename T>
T add(T a, T b) {
    return a + b;
}

Examples

1. Function Template

#include <iostream>
using namespace std;

template <typename T>
T add(T a, T b) {
    return a + b;
}

int main() {
    cout << add(5,3) << endl;
    cout << add(2.5, 3.5);
}
