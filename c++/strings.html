Strings in C++

Definition

Strings are sequences of characters. C++ supports C-style strings (char arrays) and string class.

Key Points

  • C-style strings end with '\0'.
  • String class is part of <string> library.
  • Supports operations like length, concatenation, substring, insert, erase, replace, conversion.

Syntax

C-style String

char str[size];

C++ String Class

#include <string>
string name = "Hello";

Examples (12)

1. C-style String

#include <iostream>
using namespace std;
