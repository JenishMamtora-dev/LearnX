STL in C++

Definition

The Standard Template Library (STL) provides ready-to-use classes and functions for data structures and algorithms.

Key Points

  • Includes containers (vector, list, map, etc.)
  • Provides algorithms (sort, reverse, etc.)
  • Supports iterators for traversal

Syntax (Vector)

#include <vector>
vector<data_type> v;
v.push_back(value);
v.size();
v[i];

Example

Vector Example

#include <iostream>
#include <vector>
using namespace std;
int main() {
    vector<int> v;
    v.push_back(10);
    v.push_back(20);
    v.push_back(30);
    for(int i=0;i<v.size();i++)
        cout << v[i] << " ";
}
