Data Types in C++

Definition

Data types specify the type of data a variable can store.

Key Points

  • Two Main Categories: C++ data types are divided into Primary and Derived.
  • Primary Data Types: int, float, double, char, bool — for basic value storage.
  • Derived Data Types: array, pointer, function, and reference — built from primary types.
  • User-Defined Types: struct, class, enum — custom data models.
  • Size Variation: Depends on system architecture.

Syntax

data_type variable_name;

Examples

1. Primary Data Types

#include <iostream>
using namespace std;
int main() {
    int age = 25;
    float height = 5.9;
    double pi = 3.14159;
    char grade = 'A';
    bool isCodingFun = true;

    cout << age << " " << height << " " << pi << " "
         << grade << " " << isCodingFun;
}
25 5.9 3.14159 A 1

2. Derived Data Types

#include <iostream>
using namespace std;
int main() {
    int arr[3] = {10, 20, 30};
    int* ptr = arr;
    int &ref = arr[0];

    cout << "Array first element = " << arr[0] << ", ";
    cout << "Pointer points to = " << *ptr << ", ";
    cout << "Reference = " << ref;
}
Array first element = 10, Pointer points to = 10, Reference = 10

3. User-Defined Data Types

#include <iostream>
using namespace std;
struct Point {
    int x;
    int y;
};
enum Color { RED, GREEN, BLUE };
int main() {
    Point p1 = {10, 20};
    Color c = GREEN;
    cout << "Point: (" << p1.x << "," << p1.y << "), ";
    cout << "Color = " << c;
}
Point: (10,20), Color = 1
