File Handling in C++

Definition

File handling allows reading from and writing to files stored on disk.

Key Points

  • Use fstream, ifstream, and ofstream classes.
  • Always close files after use.
  • Supports text and binary files.

Syntax

Writing to a File

#include <fstream>
ofstream fout("file.txt");
fout << "Hello";
fout.close();

Reading from a File

#include <fstream>
ifstream fin("file.txt");
fin >> variable;
fin.close();

Examples

1. Writing to a File

#include <iostream>
#include <fstream>
using namespace std;

int main() {
    ofstream fout("example.txt");
    fout << "Hello File Handling";
    fout.close();
    cout << "Data written successfully";
}
Data written successfully

2. Reading from a File

#include <iostream>
#include <fstream>
#include <string>
using namespace std;

int main() {
    ifstream fin("example.txt");
    string data;
    getline(fin, data);
    cout << "File content: " << data;
    fin.close();
}
File content: Hello File Handling
