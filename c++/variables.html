Variables in C++

Definition

A variable is a named memory location used to store data. Its value can be changed during program execution.

Key Points

  • Declaration: Every variable must be declared before use, specifying its type.
  • Memory Allocation: Memory is automatically allocated based on the variable’s data type.
  • Variable Types: Variables can be Local (inside function) or Global (outside function).
  • Initialization: Assigning a value to the variable when declaring it.
  • Constants: Declared with const keyword; value cannot be modified later.

Variable Types & Examples

1. Integer Types

int: Whole numbers, positive or negative. int age = 25;

#include <iostream>
using namespace std;
int main() {
    int age = 25;
    cout << "Age = " << age;
}
Age = 25

short: Smaller range integers. short s = 10;

#include <iostream>
using namespace std;
int main() {
    short s = 10;
    cout << "Short = " << s;
}
Short = 10

long: Larger range integers. long l = 100000;

#include <iostream>
using namespace std;
int main() {
    long l = 100000;
    cout << "Long = " << l;
}
Long = 100000

long long: Very large integers. long long ll = 1000000000;

#include <iostream>
using namespace std;
int main() {
    long long ll = 1000000000;
    cout << "Long Long = " << ll;
}
Long Long = 1000000000

2. Floating Types

float: Single precision decimals. float pi = 3.14;

#include <iostream>
using namespace std;
int main() {
    float pi = 3.14;
    cout << "PI = " << pi;
}
PI = 3.14

double: Double precision decimals. double d = 3.141592;

#include <iostream>
using namespace std;
int main() {
    double d = 3.141592;
    cout << "Double = " << d;
}
Double = 3.141592

long double: Higher precision decimals. long double ld = 3.1415926535;

#include <iostream>
using namespace std;
int main() {
    long double ld = 3.1415926535;
    cout << "Long Double = " << ld;
}
Long Double = 3.1415926535

3. Character Types

char: Single character. char c = 'A';

#include <iostream>
using namespace std;
int main() {
    char c = 'A';
    cout << "Char = " << c;
}
Char = A

wchar_t: Wide characters. wchar_t wc = L'Ω';

4. Boolean Type

bool: Stores true or false. bool flag = true;

#include <iostream>
using namespace std;
int main() {
    bool flag = true;
    cout << "Flag = " << flag;
}
Flag = 1

5. Void Type

void: Represents no value, used as function return type.

#include <iostream>
using namespace std;
void show() {
    cout << "Hello from void function!";
}
int main() {
    show();
}
Hello from void function!

6. Constant Variables

const: Value cannot be changed once assigned. const int x = 10;

#include <iostream>
using namespace std;
int main() {
    const int x = 10;
    cout << "Constant x = " << x;
}
Constant x = 10
