Pointers in C++

Definition :

A pointer is a variable that stores the memory address of another variable. It allows direct memory access and manipulation, making C++ powerful and flexible.

Key Points

  • Pointers store the memory address of another variable.
  • They allow direct access and manipulation of memory.
  • Used widely for dynamic memory, arrays, and data structures.

Types of Pointers

Key Points

  • There are several pointer types in C++ based on how they manage memory.
  • Understanding them helps prevent runtime errors and crashes.

1. Null Pointer

Key Points

  • Points to nothing (address = 0).
  • Used for safe initialization before assigning real addresses.
int *ptr = NULL;
if(ptr == NULL)
    cout << "Pointer is null";

2. Void Pointer

Key Points

  • Generic pointer type; can point to any data type.
  • Must be type-cast before dereferencing.
void *ptr;
int a = 10;
ptr = &a;
cout << *(int*)ptr;

3. Wild Pointer

Key Points

  • Declared but not initialized pointers.
  • Accessing them causes undefined behavior or crashes.
int *p;
*p = 5;  // ❌ Undefined behavior

4. Dangling Pointer

Key Points

  • Pointer pointing to memory that has been freed or deleted.
  • Accessing such pointers leads to unpredictable results.
int *p = new int(10);
delete p;
cout << *p;  // ❌ Dangling pointer

5. Constant Pointer

Key Points

  • Address stored in pointer cannot be changed.
  • Value at that address can still be modified.
int x=5, y=10;
int *const ptr = &x;
*ptr = 20;     // allowed
ptr = &y;      // ❌ not allowed

6. Pointer to Constant

Key Points

  • Data pointed to cannot be modified through the pointer.
  • The pointer itself can point to other variables.
int x = 5;
const int *ptr = &x;
*ptr = 20;   // ❌ not allowed
ptr = NULL;  // ✅ allowed

7. Constant Pointer to Constant

Key Points

  • Neither address nor data can be changed.
  • Provides complete immutability and safety.
int x = 5;
const int *const ptr = &x;
*ptr = 20;   // ❌ not allowed
ptr = NULL;  // ❌ not allowed

Examples

1. Basic Pointer

Key Points

  • '&' gives the address of a variable.
  • '*' (dereference) accesses the value stored at that address.
#include <iostream>
using namespace std;

int main() {
    int a = 10;
    int *ptr = &a;
    cout << "Value: " << *ptr << endl;
    cout << "Address: " << ptr;
}
Value: 10
Address: 0x61ff0c (example)

2. Pointer to Pointer

Key Points

  • Stores the address of another pointer.
  • Double dereference '**' is used to access the actual value.
#include <iostream>
using namespace std;

int main() {
    int x = 5;
    int *p = &x;
    int **pp = &p;
    cout << "Value: " << **pp;
}
Value: 5

3. Pointer with Array

Key Points

  • Array name acts as pointer to first element.
  • Pointer arithmetic *(p+i) is used to access array values.
#include <iostream>
using namespace std;

int main() {
    int arr[3] = {10,20,30};
    int *p = arr;
    for(int i=0;i<3;i++) {
        cout << *(p+i) << " ";
    }
}
10 20 30

4. Pointer with Function (Call by Reference)

Key Points

  • Addresses of variables are passed to function.
  • Allows modification of original variables inside function.
#include <iostream>
using namespace std;

void swap(int *a, int *b) {
    int temp = *a;
    *a = *b;
    *b = temp;
}

int main() {
    int x=10, y=20;
    swap(&x,&y);
    cout << "x=" << x << " y=" << y;
}
x=20 y=10

5. Pointer with Object

Key Points

  • '->' operator is used to access class members using pointer.
  • Pointers can refer to entire objects just like normal variables.
#include <iostream>
using namespace std;

class Student {
public:
    string name;
    Student(string n) { name = n; }
    void show() { cout << "Name: " << name; }
};

int main() {
    Student s("Rahul");
    Student *ptr = &s;
    ptr->show();
}
Name: Rahul
