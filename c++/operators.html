Operators in C++

Definition

Operators are special symbols that perform operations on operands (variables or values) to produce a result.

Key Points

  • Operators act on operands to compute results.
  • They can be classified into several types based on their functionality.
  • Each operator has a syntax and may return a value.
  • Operators can be unary (one operand), binary (two operands), or ternary (three operands).

Operator Types

1. Arithmetic Operators

Perform mathematical calculations.

Operators: +, -, *, /, %

Example:

int a = 10, b = 3;
cout << a + b; // 13
cout << a % b; // 1

2. Relational Operators

Compare two values and return true or false.

Operators: ==, !=, >, <, >=, <=

Example:

int x = 5, y = 10;
cout << (x < y); // true
cout << (x == y); // false

3. Logical Operators

Combine multiple conditions and return true or false.

Operators: &&, ||, !

Example:

bool a = true, b = false;
cout << (a && b); // false
cout << (!a);         // false

4. Assignment Operators

Assign and modify values of variables.

Operators: =, +=, -=, *=, /=, %=

Example:

int x = 10;
x += 5; // x = 15
x *= 2; // x = 30

5. Increment & Decrement Operators

Increase or decrease a variable by 1.

Operators: ++, --

Example:

int a = 5;
a++; // 6
--a; // 5

6. Bitwise Operators

Perform operations on individual bits of integers.

Operators: &, |, ^, ~, <<, >>

Example:

int a = 5, b = 3;
cout << (a & b); // 1
cout << (a | b);   // 7

7. Ternary Operator

Conditional operator that evaluates based on a condition.

Syntax: condition ? expr1 : expr2

Example:

int a = 10, b = 20;
string result = (a > b) ? "A" : "B";
cout << result; // B

8. sizeof & Comma Operators

sizeof: Returns memory size of a variable/type.

Comma: Evaluate multiple expressions and return the last one.

Example:

int a = 5, b = 10;
cout << sizeof(a);         // 4 (bytes)
int c = (a += 2, b += 3, a+b);
cout << c;                // 20
