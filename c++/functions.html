Functions in C++

Definition

A function is a block of code that performs a specific task. Functions improve code reusability, readability, and modularity.

Key Points

  • Functions allow modular programming and reduce code repetition.
  • Can return values or be void (no return).
  • Can have parameters (arguments) and default values.
  • Supports inline, recursive, overloaded, and friend functions.
  • Functions must be declared before usage (or use prototypes).

Types

  • Normal / User-defined Functions
  • Inline Functions
  • Recursive Functions
  • Function Overloading
  • Default Arguments
  • Friend Functions

Syntax

return_type function_name(parameter_list) {
    // function body
    return value; // if return_type is not void
}

Examples

1. Normal Function

#include <iostream>
using namespace std;

int add(int a, int b) {
    return a + b;
}

int main() {
    cout << "Sum = " << add(5, 3);
}
Sum = 8

2. Inline Function

#include <iostream>
using namespace std;

inline int square(int x) {
    return x * x;
}

int main() {
    cout << "Square = " << square(6);
}
Square = 36

3. Recursive Function

#include <iostream>
using namespace std;

int factorial(int n) {
    if(n == 0) return 1;
    return n * factorial(n - 1);
}

int main() {
    cout << "Factorial = " << factorial(5);
}
Factorial = 120

4. Function Overloading

#include <iostream>
using namespace std;

int area(int side) { return side * side; }
int area(int l, int b) { return l * b; }

int main() {
    cout << "Square area = " << area(5) << endl;
    cout << "Rectangle area = " << area(4,6);
}
Square area = 25
Rectangle area = 24

5. Default Arguments

#include <iostream>
using namespace std;

int power(int base, int exp=2) {
    int result=1;
    for(int i=0;i<exp;i++) result *= base;
    return result;
}

int main() {
    cout << power(5) << endl;
    cout << power(2,3);
}
25
8

6. Friend Function

#include <iostream>
using namespace std;

class Box {
private:
    int width;
public:
    Box(int w) { width = w; }
    friend void showWidth(Box b);
};

void showWidth(Box b) {
    cout << "Width = " << b.width;
}

int main() {
    Box b1(10);
    showWidth(b1);
}
Width = 10
