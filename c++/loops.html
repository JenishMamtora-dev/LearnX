Loops in C++

Definition

Loops allow repeating a block of code multiple times until a condition becomes false.

Key Points

  • Purpose: Loops help execute a set of statements repeatedly without writing them multiple times.
  • Types: C++ provides three main loops â€” for, while, and do-while.
  • Termination: Ensure conditions change to avoid infinite loops.
  • Nested Loops: A loop inside another loop for advanced tasks like patterns or matrices.

Syntax

For Loop

for(initialization; condition; increment/decrement) {
    // statements
}

While Loop

while(condition) {
    // statements
}

Do-While Loop

do {
    // statements
} while(condition);

Examples

1. For Loop Example

#include <iostream>
using namespace std;
int main() {
    for(int i=1;i<=5;i++)
        cout << i << " ";
    return 0;
}

2. While Loop Example

#include <iostream>
using namespace std;
int main() {
    int i=1;
    while(i<=5) {
        cout << i << " ";
        i++;
    }
    return 0;
}
1 2 3 4 5

3. Do-While Loop Example

#include <iostream>
using namespace std;
int main() {
    int i=1;
    do {
        cout << i << " ";
        i++;
    } while(i<=5);
    return 0;
}
1 2 3 4 5

4. Nested Loop Example

#include <iostream>
using namespace std;
int main() {
    for(int i=1;i<=3;i++) {
        for(int j=1;j<=3;j++) {
            cout << "(" << i << "," << j << ") ";
        }
        cout << endl;
    }
    return 0;
}
(1,1) (1,2) (1,3)
(2,1) (2,2) (2,3)
(3,1) (3,2) (3,3)

5. Infinite Loop Example (Avoid This)

#include <iostream>
using namespace std;
int main() {
    int i=1;
    while(true) {
        cout << i << " ";
        if(i==5) break;  // exit loop safely
        i++;
    }
    return 0;
}
