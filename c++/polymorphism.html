Polymorphism — C++

Definition

Polymorphism means "many forms". It allows same function name to behave differently.

Types

  • Compile-time: Function overloading, Operator overloading
  • Run-time: Virtual functions (method overriding)

Example

#include <iostream>
using namespace std;

class Shape {
public:
    virtual void draw() { cout << "Drawing Shape" << endl; }
};

class Circle : public Shape {
public:
    void draw() override { cout << "Drawing Circle" << endl; }
};

class Square : public Shape {
public:
    void draw() override { cout << "Drawing Square" << endl; }
};

int main() {
    Shape* s;
    Circle c; Square sq;
    s = &c; s->draw();
    s = &sq; s->draw();
}

Expected Output

Drawing Circle Drawing Square
